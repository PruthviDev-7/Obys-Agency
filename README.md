## ✨ Features

- 🎬 **Smooth Scrolling** - Locomotive Scroll for seamless navigation
- 🚀 **Advanced Animations** - GSAP powered animations
- 🎯 **Dynamic Effects** - Shery.js integration
- 📱 **Responsive Design** - Works on all devices
- 🖱️ **Mouse Effects** - Custom cursor followers
- ⚡ **High Performance** - Optimized for speed
